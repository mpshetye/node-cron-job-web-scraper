## General info
An application that runs a cron job to scrape a website
	
## Technologies
Project is created with:
* NodeJs v14.18.1
* npm v8.3.0
* chalk ^5.0.0v
* node-cron ^3.0.0v
* ora 6.0.1v
* puppeteer ^13.0.1v
* Nodemom ^2.0.15v

	
## Setup
To run this project, install it using npm:

```
$ npm install
$ npm run dev
```
or

```
$ npm install
$ npm run start
```
